## Task 1: JavaScript Placement Methods (task1_js_placement_methods.html)

This task demonstrates the three ways to include JavaScript in an HTML page:

- In the `<head>` section
- In the `<body>` section
- As an external `.js` file

**Objective:**
To understand the different methods of embedding JavaScript in HTML documents.

---

## Task 2: Addition Function (task2_addition_function.html)

This task creates a JavaScript function that adds two numbers, with both predefined and custom user input options.

### Concepts covered
- Function declaration and invocation
- `prompt()` for user input
- Type conversion with `parseFloat()`

**Objective:**
To practice writing and calling JavaScript functions with parameters.

---

## Task 3: Sum and Average (task3_sum_and_average.html)

This task prompts the user for three integers and calculates their sum and average.

### Concepts covered
- User input via `prompt()`
- Arithmetic operations
- Displaying results with `document.write()` or `alert()`

**Objective:**
To practice basic arithmetic operations and user interaction in JavaScript.

---

## Task 4: Dynamic Dropdown Lists (task4_dynamic_dropdown_lists.html)

This task dynamically populates dropdown `<select>` elements with array values using JavaScript.

### Concepts covered
- Arrays in JavaScript
- DOM manipulation
- Dynamically creating `<option>` elements

**Objective:**
To learn how to populate HTML elements dynamically using JavaScript arrays.

---

## Task 5: Student Grade Sheet (task5_student_grade_sheet.html)

This task takes marks for 5 subjects, calculates total and percentage, assigns letter grades (A-F), and displays results in a table.

### Concepts covered
- Conditional statements (`if/else`)
- Table generation with JavaScript
- Grade calculation logic

**Objective:**
To build a complete grading system using JavaScript logic and DOM manipulation.

---

## Task 6: Standard Calculator (task6_calculator.html + task6_calculator.js)

This task implements a basic calculator that performs arithmetic operations (+, -, *, /) using an external JavaScript file.

### Concepts covered
- External JavaScript files
- `prompt()` and `alert()` for I/O
- Switch/conditional statements for operator handling

**Objective:**
To practice separating JavaScript logic into external files and handling user input.

## Task 7: Arithmetic & Assignment Operators (task7_arithmetic_assignment_operators.html)

This task demonstrates arithmetic and assignment operators with step-by-step results.

### Operators demonstrated
- Arithmetic: `+`, `-`, `*`, `/`
- Assignment: `+=`, `-=`, `*=`, `/=`

**Objective:**
To understand how arithmetic and assignment operators work in JavaScript.
