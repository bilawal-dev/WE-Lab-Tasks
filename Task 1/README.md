## Task 1: Lists (lab1_lists.html)

This task demonstrates different types of lists in HTML:

- Ordered lists (`<ol>`)
- Unordered lists (`<ul>`)
- Definition lists (`<dl>`)

### Concepts covered
- Nested lists  
- Different numbering styles (A, I, etc.)  
- Combination of ordered, unordered, and definition lists  

**Objective:**  
To understand how structured and hierarchical data is represented in HTML.

## Task 2: Registration Form (lab2_registration_form.html)

This task implements a complete registration form using HTML tables and form controls.

### Features included
- Text, email, password, and URL inputs  
- Radio buttons and checkboxes  
- Dropdown menu (`<select>`)  
- Textarea for additional information  
- Submit and reset buttons  
- Table-based layout for structured form design  

**Objective:**  
To practice building structured forms and working with different HTML input types.

## Task 3: HTML5 Elements (lab3_html5_elements.html)

This task focuses on modern HTML5 features.

### HTML5 elements demonstrated
- Color picker (`type="color"`)  
- Number input with restrictions  
- Date picker  
- Audio player (`<audio>`)  
- Canvas graphics (`<canvas>` with JavaScript)  

**Objective:**  
To explore new HTML5 form controls and multimedia/graphics support.
