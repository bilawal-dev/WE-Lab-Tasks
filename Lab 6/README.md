# Lab 6 - Hoisting, Arrow Functions, Callbacks, Promises, and Async/Await

## Tasks

### Task 1 - Hoisting for let, const and var
Explore how hoisting works differently for `var`, `let`, and `const` declarations, including the Temporal Dead Zone.

### Task 2 - Arrow Function Display Name
Write an arrow function that displays your name.

### Task 3 - Arrow Add Function
Convert a simple add function that takes two numbers and returns their sum to an arrow function.

### Task 4 - Callback processData
Implement a function processData that receives data and logs it. Pass this as a callback to getData.

### Task 5 - fetchUserData with Callback
Write a function fetchUserData that takes a callback and simulates fetching user data after 10 seconds.

### Task 6 - Promise getData
Rewrite the getData function to return a promise instead of using a callback. Chain then() and catch() to it.

### Task 7 - Async/Await getData
Convert the promise chain from Task 6 into an async function using await. Add error handling using try/catch.
