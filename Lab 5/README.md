# Lab 5 - JavaScript Objects, Events, and Form Validation

## Tasks

### Task 1 - Object Key-Value Display
Create an object having first name, last name and age and display the object key and its associated value in a table.

### Task 2 - Addition of Two Numbers
Three text boxes where the user provides integer numbers in the first two and the third shows the addition result. When only the first textbox has input, the result shows NaN. After providing input in the second textbox, the result is displayed.

### Task 3 - Dropdown to Textbox Replacement
A dropdown menu with a "New Entry" button. When the button is clicked, the dropdown is replaced with a textbox.

### Task 4 - Sign Up Form Validation
Apply JavaScript validation for an HTML sign up form. Show red color error messages against each invalid input field on form submission.

### Task 5 - Display Selected Text using onSelect Event
A textbox where the user enters text, selects some or all of it, and the selected text is displayed in a separate div.
